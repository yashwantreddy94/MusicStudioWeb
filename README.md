# 🎵 Music Studio Website

A modern and responsive website for a Record Label & Music Studio, designed to showcase artists, music releases, studio services, events, and brand identity. The project provides a professional online presence for music production companies, recording studios, independent artists, and record labels.

## 🌐 Live Demo

**Website:** https://yashwantreddy94.github.io/MusicStudioWeb/

## ✨ Features

* Modern and responsive design
* Artist showcase section
* Music release and album promotion
* Studio services presentation
* Event and performance highlights
* Contact and inquiry section
* Mobile-friendly user interface
* Fast-loading static website

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap
* Font Awesome
